# LabInsight 건강검진 서비스 데모

GitHub Pages에서 바로 실행할 수 있는 정적 웹사이트입니다.

## 목적

LabInsight는 건강검진 결과를 단순히 저장하는 서비스가 아니라, 검사 수치를 이해 가능한 언어로 해석하고 다음 행동으로 연결하는 건강검진 결과 해석 서비스입니다.

## 현재 구현 범위

- 건강검진 기본 정보 입력
- 결과지 파일 선택 UI
- 간기능, 신장·대사, 지질, 혈액 항목 직접 입력
- 참고 범위 기반 정상·주의·위험 분류
- 건강관리 위험도 점수 계산
- 이상 항목 요약
- 생활 가이드 및 진료과 연계 후보 표시
- 가족 단위 건강 이력 관리/서비스 확장 로드맵 섹션

## 현재 버전의 한계

- 실제 OCR은 아직 연결되어 있지 않습니다.
- 실제 병원 예약/가격 비교 API는 연결되어 있지 않습니다.
- 기준치는 교육용 참고 범위입니다.
- 실제 진단, 치료, 처방 판단에는 사용할 수 없습니다.
